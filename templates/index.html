Bus Departures

About

West Yorkshire Metro provide a "yournextbus" service however you can only search bus departures for one stop at a time. With this tool you access bus times for multiple stops.

This is not an ideal soloution, it involves making a seperate request for each bus stop. This means that bus stations with a lot of stops eg. Leeds City will take a good few seconds to load.
